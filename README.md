This project should be run in the following order:

##1. Prelim.ipynb file (Data Cleaning)

  Run the Prelim notebook first.
  This step cleans and prepares the raw data for modeling.
  Make sure the cleaned output file is successfully generated before moving on. This will be named cleaned_flight_data.csv

##2. Modeling Step After data cleaning is complete, choose one of the following paths:

  ###a. Supervised Learning file (SupervisedLearning.ipynb)

  Run this file if you want to execute our supervised learning model. 
  
  ###b. Unsupervised Learning file (UnsupervisedLearning.ipynb)

  Run this file if you want to execute our unsupervised learning model.
